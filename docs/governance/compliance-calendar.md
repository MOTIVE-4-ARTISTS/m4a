## Threshold triggers (the filing changes when we cross these)

- Gross receipts **> $50k** → Form **990-EZ** replaces 990-N.
- Gross receipts **> $200k** or assets **> $500k** → full Form **990** required.
- Gross receipts **> $1M** → NY CHAR500 **independent audit** requirement.
- Any **lobbying** expense → consider the **501(h) election** (Form 5768).
